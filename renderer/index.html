Wie funktioniert das?
Der Entwickler veröffentlicht eine neue Version als GitHub Release. Die App prüft dort auf latest.yml, lädt den neuen Installer herunter und installiert ihn beim nächsten Start. Der Benutzer muss nichts manuell herunterladen.

Wie funktioniert das?
