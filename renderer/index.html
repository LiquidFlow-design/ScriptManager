🖥️ App-Update — neue .exe Version
Lädt eine neue Version der App herunter und installiert sie. Erfordert eine neue GitHub Release vom Entwickler.
Installierte Version
–
Verfügbare Version

Bereit. Klicke "Prüfen" um nach einer neuen App-Version zu suchen.
Wie funktioniert das?
Der Entwickler veröffentlicht eine neue Version als GitHub Release. Die App prüft dort auf latest.yml, lädt den neuen Installer herunter und installiert ihn beim nächsten Start. Der Benutzer muss nichts manuell herunterladen.
📜 Script-Update — PS1-Dateien via Git
Aktualisiert nur die PowerShell-Scripte aus dem GitHub-Repo. Kein Neustart, keine neue App-Version nötig.
Lokaler Stand
–
Remote Stand
–
Bereit. Klicke "Status prüfen" um nach neuen Scripten zu suchen.
Wie funktioniert das?
Die PS1-Scripte liegen in %AppData%\PSScriptManager\lib\. Beim ersten Sync wird das Repo geklont, danach läuft ein git pull. Neue Scripte werden automatisch erkannt. Git muss installiert sein.
